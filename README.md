# 📚 Smart Study Planner

A Django-based web application that helps students manage their study schedule, track exams, and generate personalized timetables based on availability.

---

## 🚀 Features

* 📅 Weekly Study Timetable
* 🤖 AI-based Timetable Generation
* 🧠 Subject & Study Planning
* 📝 Exam Scheduling & Tracking
* 👤 User Authentication System
* 📊 Dashboard Overview

---

## 🛠 Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **Authentication:** Django Auth System

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/smart-study-planner.git
cd smart-study-planner
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run migrations

```
python manage.py migrate
```

### 5️⃣ Start server

```
python manage.py runserver
